# 🚀 InterviewIQ – AI Resume Analyzer & Interview Prep

## 📌 Overview

InterviewIQ is a full-stack AI-powered platform that helps users:

* Analyze resumes
* Identify skill gaps
* Practice interview questions
* Improve job readiness

## ⚙️ Installation

### Clone repo

```bash
git clone https://github.com/ambuj25-crypto/InterviewIQ.git
```

### Backend setup

```bash
cd backend
npm install
npm start
```

### Frontend setup

```bash
cd frontend
npm install
npm run dev
```
